## 设计亮点

### 1. 视觉层次清晰
- 使用图标容器 (`w-10 h-10 rounded-lg bg-primary/10`) 增强视觉识别
- 配置项使用不同背景色区分状态

### 2. 交互反馈良好
- Hover 状态有明显视觉反馈
- 保存按钮有加载状态
- Toast 通知用户操作结果

### 3. 信息架构合理
- 全局设置集中在一个卡片
- 每个配置项有清晰的标题和描述
- 触发节点配置仅在启用时显示（渐进式披露）

### 4. 颜色使用恰当
- 蓝色 - 信息/创建
- 绿色 - 完成/成功
- 琥珀色 - 警告/注意
- 紫色 - 集成/推送
